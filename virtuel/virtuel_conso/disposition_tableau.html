Affichage en disposition tableau dans un virtuel

  • Installer le plugin Virtuel : Disponible sur le ➔ Market Jeedom
  • Accéder au plugin Virtuel : Sur la barre du haut, cliquez sur l'onglet   Plugins ➔ Programmation ➔ Virtuel

  • Signification: L'affichage en mode tableau sur Jeedom signifie généralement la présentation des données dans une interface que l'on peut personnaliser en fonction de ses besoins spécifiques.
  • Widget:  un widget est une interface graphique qui affiche des informations ou permet l'interaction avec des équipements domotiques. Lorsque l'on parle d'un affichage en mode tableau, on se réfère à une disposition de données organisées sous forme de tableau, où les informations sont présentées dans des lignes et des colonnes.
  • En résumé:  l'affichage en mode tableau d'un widget sur Jeedom implique la présentation structurée de données dans une interface, avec la possibilité de personnaliser l'apparence et le style grâce à des fonctionnalités telles que l'utilisation de CSS (Cascading Style Sheets). Cela permet une meilleure visualisation et interaction avec les informations.

  • Accès à la configuration :

    • Cliquez sur le virtuel précédemment créé et cliquez ensuite sur   Configuration avancée ➔   Disposition➔ sélectionnez tableau

  • Remarque : Vous constaterez que j'ai sélectionné 1 ligne et 4 colonnes

    • Colonne 1: Affichage de l'icône  Colonne 2: Affichage du nom de l'équipement  Colonne 3: Espace  Colonne 4: Affichage de la valeur de communication

  • Positionnement des commandes :

    • Le « Drag and Drop » ou « Glisser/Déposer » est une technique qui permet de déplacer un élément dans une interface graphique



  • CSS:

    • Les propriétés CSS peuvent bien sûr être modifiées comme bon vous semble

    Voici les miennes
